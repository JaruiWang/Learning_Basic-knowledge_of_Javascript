## JavaScript 输入输出语句
- prompt(info) 浏览器弹出输入框，用户可以输入内容
```JS
    <script>
        // 这是一个输入框
        prompt("请输入您的年龄");
    </script>

    // 运行效果查看prompt.JPG
```

- alert(msg)  浏览器弹出警示框
```JS
    <script>
        alert("我的年龄是23岁");
    </script>
     // 运行效果查看alert.JPG
```

- console.log(msg)  浏览器控制台打印输出信息
控制台输出  给程序员测试用的 
console 控制台   log 日志，记录
